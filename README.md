# Beskrivning av projektet
BookReview är en webbaserad plattform utvecklad för att underlätta sökning, recension och upptäckt av litteratur. Applikationen syftar till att lösa problemet med ostrukturerade recensionssidor genom att erbjuda en samlad plats där användargenererat innehåll kompletteras med AI-stöd.
Många användare upplever idag att det saknas ett enkelt sätt att hitta bra böcker och att befintliga plattformar ofta är röriga. Dessutom saknas verktyg som effektivt hjälper användaren att formulera egna omdömen.

BookReview adresserar detta genom att kombinera vanliga recensioner med AI-genererade insikter. Systemet är uppbyggt med en tydlig struktur för böcker, användare och recensioner, vilket gör det enkelt att navigera och bidra med innehåll.
Syftet med projektet är att skapa ett användarvänligt verktyg där både AI och människor kan bidra med recensioner på samma bok. Målet är att sänka tröskeln för att skriva recensioner genom inbyggd AI-assistans och att tillhandahålla en modern, sökbar databas för böcker
